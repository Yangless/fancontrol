## 实验记录模板

每次实验请使用以下模板:

```markdown
# 实验: <配置名称>

**开始时间**: YYYY-MM-DD HH:MM
**结束时间**: YYYY-MM-DD HH:MM

## 修改参数

| 参数 | 原值 | 新值 | 变化幅度 |
|------|------|------|---------|
| IdleTemperature | 35 | 40 | +14% |
| MinFanSpeed | 30 | 25 | -17% |

## 测试条件

- **负载场景**: 空载 / 游戏 / 压力测试
- **环境温度**: XX°C
- **测试时长**: XX 分钟

## 测试结果

| 指标 | 基准 | 实验 | 变化 |
|------|------|------|------|
| 平均温度 | XX°C | XX°C | +/-X.X°C |
| 峰值温度 | XX°C | XX°C | +/-X.X°C |
| 平均转速 | XXXX RPM | XXXX RPM | +/-XXX RPM |
| 噪音评分 | X/10 | X/10 | +/-X |

## 决策

- [ ] ✅ 接受
- [ ] ❌ 拒绝
- [ ] ⚠️ 需进一步测试

**理由**: <填写决策依据>

## 后续行动

- <如果接受: 合并到主配置>
- <如果拒绝: 尝试其他参数组合>
```
